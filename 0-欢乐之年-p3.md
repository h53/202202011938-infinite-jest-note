# 欢乐之年

我坐在一间办公室里，周围环绕着头脑和身躯。我有意识地将姿势贴合硬椅子的形状。这是一间寒冷的大学行政室，木墙，[^Remington-hung]，离11月的高温相隔两扇窗户，远离行政室外面接待区的声音，刚刚我和查尔斯叔叔以及德林特先生就是在那儿被迎接的。

我就在这里。

视线穿过闪耀着亚利桑那州午时[^蜘蛛灯光]的抛光松木会议桌，在夏季运动外套和半温莎结之上，我看到三张脸。这是三位院长 —— 负责招生，学术以及体育事物。我不知道哪张脸属于哪个人。

我相信我表现的中立，甚至可能愉悦，尽管我因犯过错而受训要站在中立的一边，且不要表现出愉悦或微笑的意图。

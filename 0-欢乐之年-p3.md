# 欢乐之年

我坐在一间办公室里，周围环绕着头脑和身躯。我有意识地将姿势贴合硬椅子的形状。这是一间寒冷的大学行政室，木墙，Remington-hung[^Remington-hung]，离11月的高温相隔两扇窗户，远离行政室外面接待区的声音，刚刚我和查尔斯叔叔以及德林特先生就是在那儿被迎接的。

我就在这里。

视线穿过闪耀着亚利桑那州午时蜘蛛灯光[^蜘蛛灯光]的抛光松木会议桌，在夏季运动外套和半温莎结之上，我看到三张脸。这是三位院长 —— 负责招生，学术以及体育事物。我不知道哪张脸属于哪个人。

我相信我表现的中立，甚至可能愉悦，尽管我因犯过错而受训要站在中立的一边，且不要表现出愉悦或微笑的意图。

我已经小心地翘起二郎腿，至少我希望如此，脚踝放在膝盖，双手放在长裤上。我的手指互相交叉成为一个个镜像的字母“X”。这间会议室的其他人包括：大学文学主管，学校网球代表队教练，以及我旁边的学院副院长德林先生；在我视线边缘的座位上分别地站着一些人。网球教练的零钱叮当作响。房间里有什么东西依稀发酵出味道。免费赠送的高牵引力鞋底的耐克跑鞋，和我妈妈同父异母/同母异父的兄弟的乐福鞋一起摇晃着，作为校长，他坐在椅子的正中间，像我希望的那样，并且也面朝迪恩斯。
